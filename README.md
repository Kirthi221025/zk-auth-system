 zk-auth-system

A Django-based Zero-Knowledge (ZK) authentication system.

Features

- ZK Challenge generation and verification
- JWT-based authentication
- Admin panel and dashboard
- Backend API ready for frontend integration

Tech Stack

- Python 3.x
- Django 6.x
- SQLite (dev)
- React / Frontend (separate repo)

 Setup

1. Clone repo: git clone https://github.com/Kirthi221025/zk-auth-system.git
2. Create virtualenv: python -m venv venv
3. Install requirements: pip install -r requirements.txt
4. Run server: python manage.py runserver
